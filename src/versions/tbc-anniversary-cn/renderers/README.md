# Renderers

Renderer scripts convert `layouts/*.json` into addon SavedVariables.

Planned renderers:

- Bartender4
- Shadowed Unit Frames
- Masque
- Questie tracker positioning
- Damage/threat meter positioning

Current implemented renderer:

```text
scripts/render-layout.py
```

```text
configs/tbc-anniversary-cn/Autyan/
```
